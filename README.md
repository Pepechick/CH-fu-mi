# CH-fu-miPour jouer dans la partie, chaque joueur appuie 3 à 4 secondes sur A.
Puis, si celui-ci s’est bien connecté à la partie, un "C" va s’afficher pour l’informer qu’il est bien connecté.

Ensuite, il devra choisir son pseudo : juste après que le C soit apparu, des lettres s’affichent. Il devra donc les faire défiler en appuyant sur B 
pour choisir son pseudo, puis appuyer sur A dès qu’il veut ajouter une lettre.

Une fois le pseudo choisi, le joueur devra faire un appui long de 3 à 4 secondes pour valider le pseudo et entrer dans la partie.
